GONZALEZ, R; WOODS, R; EDDINS, S. Digital Image Processing Using MATLAB 2. 2. ed. Gatesmark Publishing, 2009. Mathworks Stack Exchange Wikipedia

Descrição:

Utilizando a imagem da lua de Jupiter, 1. segmente as regiões de ‘cor avermelhada’. 2. visualize e comente arquivo: Fig0642(a)(jupiter_moon_original).tif

Pacotes

pkg load signal;
pkg load symbolic;
pkg load image;

Exibir imagem

rgb = im2double(imread("Fig0628(b)(jupiter-Io-closeup).tif"));
[M, N, ~] = size(rgb);
figure;
  imshow(rgb,[]);
  title("Imagem original");

Conversão para HSV

HSV = rgb2hsv(rgb);
figure;
  imshow(HSV,[]);
  title("Imagem original em HSV");

Decomposição da imagem

H = HSV(:,:,1);
S = HSV(:,:,2);
V = HSV(:,:,3);
figure;
  imshow(H,[]);
  title("Matiz");
figure;
  imshow(S,[]);
  title("Saturação");
figure;
  imshow(V,[]);
  title("Intensidade");

Máscara de saturação

sbw = double(im2bw(S,0.35));
figure;
  imshow(sbw,[]);
  title("Máscara de saturação");

Multiplicar a máscara de saturação pela matiz

hue = H.*sbw;
figure;
  imshow(sbw,[]);title("Máscara de saturação");
figure;
  imshow(H,[]); title("Matiz");
figure;
  imshow(hue,[]);
  title("Máscara de saturação .* matiz");

Limiarização para destcar regiõe avermelhadas

hue2 = zeros(M,N);
hue2(hue>0.5)=1;
figure;
  imshow(hue2,[]);
  title("Regiões com cores vermelhas");
